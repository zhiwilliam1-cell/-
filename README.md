我构建了一个基于多智能体协同的法律合同智能体系统，核心解决传统合同审查效率低、长文本处理困难、法律依据不可追溯及敏感信息泄露风险等痛点。系统以主从式多 Agent 架构为核心：主智能体负责任务分配、结果汇总、去重与风险排序，法律审查、财务审查、基础规范审查三个子智能体并行工作，分别处理合规性、金额付款、格式语法等不同维度的问题。系统还引入 Agentic RAG 智能问答模块，通过查询规划、向量检索与 BM25 混合检索、重排序和引用溯源，实现合同内容多轮自然语言问答。开发过程中结合 Claude Code 按 Spec 文档驱动实现，并通过 Hooks 对关键流程进行校验和记录，提升开发可控性与可复现性。同时系统设计了 PII 隐私脱敏与 Human-in-the-Loop 人工确认机制，保证合同敏感信息安全，并将最终决策权保留给人工。项目已完成前后端系统、规则管理、合同审查、风险高亮定位、智能问答和溯源展示等功能，并获得学院认可计划本月内在校内部署上线。测试结果显示，合同风险识别召回率达到 0.859，问答模块 RAGAS 四项指标均超过 0.76，审查平均响应时间为 318.9 秒，问答平均响应时间约 12.79 秒。
